# IstioCircuitBreaker
This project will help you setup a [kind](https://kind.sigs.k8s.io/) cluster with [istio](https://istio.io/), [grafana](https://grafana.com/), [k6](https://k6.io/) and custom a app using [task](https://taskfile.dev/).

## prerequisites 
- [kind](https://kind.sigs.k8s.io/)
- [docker](https://www.docker.com/)
- [task](https://taskfile.dev/)
- [helm](https://helm.sh/)

### optional (if you want to change the app)
- [go](https://go.dev/)
