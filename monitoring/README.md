# Monitoring

This directory contains everything related to monitoring the Kubernetes cluster using Grafana.

## Contents

- **Grafonnet**: A set of [Jsonnet](https://jsonnet.org/) files used to define and create Grafana dashboards in a programmatic way.
- **Grafana Manifests**: Kubernetes manifests used to deploy and configure Grafana in the cluster.

These files allow you to set up Grafana with custom dashboards tailored to your needs, providing powerful visualizations for monitoring the cluster and application performance.
